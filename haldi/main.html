My Application at Haldi


• What attracted you to Haldi and this role?

I recently completed my Bachelor's degree in Computer Science and I am actively looking for software engineering roles for new grads. I got attracted to Haldi specially because of the name which means turmeric to Indian people and by seeing how two women founded Haldi. Every women is interested in getting a great skincare routine, and for me ,a women in tech getting a skincare and then working for the brand would be a great opportunity to start my career in.


• Tell us about a software project that you worked on -- What was the problem you were trying to solve? What did you build? What was an issue or challenge you ran into along the way and how did you solve it?

One of my recent project that I worked is from my Software Engineering coursework. We build a Java Pong Application/Game using Java, Object-Oriented Programming and JavaFx. It was a collaborative project of 25 students . We had a basic pong game, and then each student had their own better and modified version of the game. My version of the game included adding some background music, making the UI better , changing speed of the puck and some other features. The only challenge was at that time I was new to Java Programming, so I had to get comfortable in Java in order to complete the application. But, overall I gained a really great experience of JUnit testing and Object-Oriented Programming.
